Run ML on AMD GPU

1. Windows version (>10, 1709)
2. Install minconda in C:\Users\amitk\
What is conda? conda is package manager which will install python,other required packages.
3. Create python env using conda, Activate

//Create a new environment named ml_venv, install Python 3.6
cmd>
C:\Users\amitk\source\repos\Python\ML>C:\Users\amitk\miniconda3\Scripts\conda create --name ml_venv python=3.6
C:\Users\amitk\source\repos\Python\ML>"c:\Users\amitk\miniconda3\Scripts\activate" ml_venv
//Env is created here: C:\Users\amitk\miniconda3\envs
(ml_venv) C:\Users\amitk\source\repos\Python\ML>"c:\Users\amitk\miniconda3\condabin\deactivate.bat"  //deactivate
(ml_venv) C:\Users\amitk\source\repos\Python\ML>
4. Install the TensorFlow with DirectML package
(directml) C:\Users\amitk\source\repos\Python\ML>pip install tensorflow-directml
5. Add 2 tensors on GPU
